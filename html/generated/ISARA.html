Retr_CRI(wvl, measured_sca_coef, ...)

Returns aerosol particle real and imaginary refractive index from three scattering coefficeint measurements, three absorption coefficient measurements, a measured number concentration for an aerosol size distribution.

Retr_kappa(wvl, measured_wet_sca_coef, sd1, ...)

Returns aerosol particle hygroscopic growth factor from three dry scattering coefficeint measurements, three humidifide scattering coefficient measurements, a measured number concentration for an aerosol size distribution.

Exceptions

InvalidNumberOfWavelengths

Raised if the length of wvl is not 6.
